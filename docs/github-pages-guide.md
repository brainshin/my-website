# GitHub Pages 배포 가이드

GitHub Pages로 정적 사이트를 배포할 때 확인할 항목을 정리합니다.

## 기본 흐름

| 단계 | 설명 | 확인 명령 또는 위치 |
| --- | --- | --- |
| 저장소 준비 | Pages로 배포할 저장소를 준비합니다. | GitHub repository |
| 파일 작성 | `index.html` 또는 `index.md`를 작성합니다. | repository root |
| push | `main` 브랜치에 변경사항을 올립니다. | `git push origin main` |
| Pages 확인 | Pages Actions가 성공했는지 봅니다. | Actions 탭 |
